# ShowLetters.exe

- 顯示選定之Windows安裝字形，以檢查是否超出預設字形邊框
- U8g2FontTools.exe(v1.0.23.512)以預設邊框為基礎進行編碼，超出之純英文字字形可能被裁切
- 新版U8g2FontTools.exe將儘可能修正被裁切錯誤

程式設計：林威辰(臺灣.高雄)

程式語言：Embarcadero C++ 11 Alexandria

完成日期：2023-05-14
